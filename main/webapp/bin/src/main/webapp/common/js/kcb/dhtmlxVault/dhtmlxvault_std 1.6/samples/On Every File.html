DHTMLX Vault sample

On Every File

You can place this JavaScript file upload anywhere on your web page, attaching it to any div object.
Set up server handlers for your server type.

Click "Add" button to select files.

<div id="vaultDiv"></div> <script> vault=new dhtmlXVaultObject(); vault.setImagePath("codebase/imgs/"); vault.setServerHandlers("UploadHandler.php", "GetInfoHandler.php", "GetIdHandler.php"); vault.onFileUploaded = function(file) { alert("id:" + file.id + ",name:" + file.name + ",uploaded:" + file.uploaded + ",error:" + file.error); }; vault.create("vaultDiv"); </script>
